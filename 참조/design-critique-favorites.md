# 디자인 크리틱: 즐겨찾기함 (branddesign-favorites.vercel.app)

## 전체 인상
사내 링크(코나베이스, BX업무 등)와 개인 링크(뉴스, AI 도구 등)를 카테고리별로 모아두는 북마크 대시보드. 카드 스타일은 깔끔하고 일관되지만, 고정(sticky) 툴바가 콘텐츠와 겹치고 데스크톱 폭에서 화면 오른쪽 절반이 비어있는 게 가장 큰 개선 포인트다.

## 사용성

| 항목 | 심각도 | 권장사항 |
|---|---|---|
| 우측 상단 고정 툴바(☀️ / +내 링크 / ⋮)가 스크롤해도 항상 같은 위치에 떠 있어 "사내" 섹션 첫 카드, 검색창 우측과 겹침 | 🟡 Moderate | 툴바에 배경 블러+그림자를 더 강하게 주거나, 스크롤 시 살짝 투명해지게 하거나, 카드 쪽에 우측 패딩을 더 줘서 물리적으로 겹치지 않게 조정 |
| 데스크톱 폭에서 콘텐츠가 화면 좌측 절반(약 530px)에만 표시되고 오른쪽은 빈 배경 | 🟡 Moderate | 넓은 뷰포트에서는 2~3열 그리드로 전환하거나, 최대 폭을 유지하되 페이지를 중앙 정렬해 여백을 좌우 대칭으로 |
| "편집 모드" 안내 문구가 상시 노출되어 있고 본문처럼 흘러가 눈에 잘 안 띔 | 🟢 Minor | 툴팁이나 접을 수 있는 배너로 전환, 최초 1회만 노출 |

## 시각적 위계

- **처음 시선이 가는 곳**: 페이지 로드 시 상단에 큰 여백만 보이고 실제 콘텐츠(탭, 검색, 카드)는 스크롤해야 보임 — 로딩이 덜 된 것처럼 보일 위험
- **읽기 흐름**: 카테고리 탭 → 검색 → 카드 리스트 순서는 논리적이고 좋음
- **강조**: 카테고리명(사내, 업무 등)이 본문 카드 제목과 대비가 약해 섹션 구분이 다소 흐릿함 — 섹션 라벨에 더 진한 색이나 상단 구분선 추천

## 일관성

| 요소 | 이슈 | 권장사항 |
|---|---|---|
| 아이콘 스타일 | 사내 링크는 통일된 빨간 "K" 사각 아이콘인데, 외부 링크는 실제 favicon(늑대 로고, "H" 로고, 텍스트 로고 등)이 혼재해 톤이 들쭉날쭉 | 외부 favicon도 동일한 라운드 사각 프레임 + 얇은 테두리로 감싸 톤을 맞추면 카드 전체가 더 정돈되어 보임 |
| 카드 간격/그림자 | 카드끼리 그림자 톤은 일관됨 | 유지 |

## 접근성

- **색 대비**: URL/부제 텍스트가 연한 회색이라 밝은 배경에서 WCAG AA 기준(4.5:1) 근처이거나 미달일 가능성 있음 — 실제 hex 값 확인 후 필요시 한 단계 더 진하게 조정 권장
- **터치 타겟**: 카드, "보드 편집"/"숨긴 즐겨찾기" 버튼 모두 충분한 크기로 양호
- **텍스트 가독성**: 제목/부제 폰트 크기 차이가 적절해 읽기 편함

## 잘 되어 있는 점

- 카드 하나하나의 정보 구조(아이콘 + 제목 + 설명 + URL)가 명확하고 스캔하기 쉬움
- 카테고리 탭으로 필터링하는 구조가 북마크 개수가 많아져도 확장 가능

## 우선순위 권장사항

1. **고정 툴바와 콘텐츠 겹침 해결** — 실사용 중 클릭 오작동/시야 가림으로 이어질 수 있는 가장 실질적인 버그성 이슈
2. **데스크톱 레이아웃 대응** — 현재는 모바일 폭 그대로 데스크톱에 얹어놓은 형태라 넓은 화면에서 어색함
3. **외부 링크 아이콘 톤 통일** — 브랜드 일관성 측면에서 비교적 쉽게 개선 가능
